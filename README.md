# ESTA渡航認証サイト

米国ESTA（電子渡航認証システム）の申請ガイドサイト

## 🎯 プロジェクト概要

K-ETA（韓国）とETIAS.co.jp（欧州）のデザインを参考に、日本人向けESTA申請ガイドサイトを構築しました。

### ページ構成
- **トップページ** (`/`)
  - ヒーローセクション
  - 3つの重要情報（申請対象、手数料、審査時間）
  - ESTAとは？（概要）
  - 3ステップ申請プロセス
  - よくある3つの質問
  - 詐欺サイト警告
  
- **申請手順ガイド** (`/application-guide`)
  - 申請前の準備
  - ステップ1-8の詳細ガイド
  - よくある記入ミス TOP5
  
- **費用比較** (`/cost-comparison`)
  - 公式サイト vs 代行サービス比較表
  - メリット・デメリット
  - 診断ツール（どちらを選ぶべきか）
  - 詐欺サイト警告
  
- **FAQ** (`/faq`)
  - 6カテゴリ、25問以上のQ&A
  - 基本情報、申請プロセス、費用、トラブル、渡航、特別なケース

## 🚀 開発環境のセットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

### 3. ビルド（静的エクスポート）

```bash
npm run build
```

静的ファイルは `out/` ディレクトリに出力されます。

### 4. デプロイ

詳細は `DEPLOYMENT.md` を参照してください。

**Cloudflare Pages設定:**
- ビルドコマンド: `npm run build`
- 出力ディレクトリ: `out`
- Node.jsバージョン: 18以上

## 🎨 デザインの特徴

### K-ETAから学んだポイント
- シンプルなヒーローセクション
